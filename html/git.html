Git

Базовые команды для работы:

  • GIT INIT - Создает новый локальный репозиторий.
  • GIT CLONE - Клонирует существующий удаленный репозиторий.
  • GIT STATUS - Показывает состояние рабочей директории (измененные файлы).
  • GIT ADD [file] - Добавляет файлы в область подготовки (staging area).
  • GIT COMMIT -M "Сообщение" - Фиксирует изменения в истории.

Удаленные репозитории:

  • GIT PUSH - Отправляет локальные коммиты на удаленный сервер.
  • GIT PULL - Получает изменения с удаленного сервера и объединяет с локальными.
